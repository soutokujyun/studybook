## 欧几里得算法
求最大公约数
终极奥义 gcd(a,b) = gcd(b, a%b) = gcd(b, 0)
```
function gcd(a, b) {
  if (b) gcd(b, a % b) 
  return a;
}
```

## 数论中的欧拉公式
欧拉函数φ(n)

φ(n) 小于n且与n互质数的个数

6 = 2 * 3
φ(6) = 6 - 6/3 - 6/2 + 6/(2*3)

φ(12) = 2^2 * 3^1
        = 2^1 * 3^0 * (2 - 1) * (3 -1)
        = 4

φ(n) = (a^x, b^y) = a^(x-1)·(a-1)·b^(y-1)·(b-1)


欧拉公式 a^φ(n) mod n = 1

### 意义
a^0 = 1   a^φ(n) = 1
a^1 = r1  a^(φ(n)+1) = r1
a^2 = r2  a^(φ(n)+2) = r2
有循环解， 循环解 = φ(n)

x·i <> x·j => a·xi <> a·xj

> 证明
S =  {x1,x2,x3...x^φ(n)}
S` = {ax1, ax2, ax3...ax^φ(n)}

(x1·x2·x3·...·x^φ(n)) = a^φ(n) * (x1·x2·x3·...·x^φ(n)) mod n;
1 = a^φ(n) mod n;
