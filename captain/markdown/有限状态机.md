# 有限状态机 --FSM
阮一峰的 有限状态机

有限状态机表示有限个状态以及在这些状态之间的**转移和动作**等行为计算模型

分为2个大类：
1. 若输出只和状态有关而与输入无关，则成为Moore状态机。
2. 输出不仅和状态有关而且和输入有关系，则称为Mealy状态机。
