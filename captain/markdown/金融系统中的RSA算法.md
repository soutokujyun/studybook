## RSA 算法过程

### 如何做钥匙
1. 选择两块上好的【铁皮】
N = P * Q
2. 做出【钥匙】的模具
Φ(n) = (P - 1) * (Q - 1)
3. 随便做一把【加密】的【钥匙】
选择 e 与 Φ(n) 互质即可（最大公约数等于1）
4. 根据【加密钥匙】精心设计一把【解密钥匙】
e * d = 1 (mod Φ(n)) => e * d % Φ(n) = 1
5. 打包【加密钥匙】和【解密钥匙】
【加密钥匙】：(e, N)  --> 公钥
【解密钥匙】：(d, N)  --> 私钥

## 如何使用两把【钥匙】
1. 加密人使用【加密钥匙】(e, N)
加密过程：M^e % N = C
解密过程：C^d % N = M

## 非对称加密的好处
被加密的方法和信息都可以随意被窃取，反正你没有解密方法
