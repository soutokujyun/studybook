# 可空类型
## 单问好 ? 与 双问号 ??
1. ? 单问号用于对 int，double,bool等无法直接赋值为null的数据类型进行null的赋值，意思是这个数据类型是Nullable类型。
```csharp
int? i = null;
```
2. ?? 用于判断一个变量在为null的时候返回一个指定的值。
```csharp
int? i;
i = i ?? 0.5;
```

## 可控类型
是在基础类型的取值范围上，再加上一个null值。
例如：Nullable<bool>变量可以被赋值为true，false，null。
声明一个nullable类型的语法如下：
```csharp
<data_type> ? <variable_name> = null;
```
